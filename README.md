# cpu-affinity

CPU affinity for processes and threads across many platforms and Operating Systems, including:-

* Android
* BitRig (does nothing)
* DragonFlyBSD
* Emscripten
* Fuschia
* FreeBSD
* iOS
* MacOS (does nothing, but special logic for setting thread affinity groups)
* Linux
* NetBSD
* OpenBSD (does nothing)
* Windows
